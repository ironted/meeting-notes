Fractals!




What are fractals?

The term "fractal" was first coined by Benoit Mandelbrot in 1975 based on the Latin frāctus, meaning "broken" or "fractured"

any of various extremely irregular curves or shapes for which any suitably chosen part is similar in shape to a given larger or smaller part when magnified or reduced to the same size

- Marriam-Webster

beautiful, damn hard, increasingly useful. That's fractals - Benoit Mandelbrot

A fractal is by definition a set for which the Hausdorff–Besicovitch dimension strictly exceeds the topological dimension - Benoit Mandelbrot

A fractal is a shape made of parts similar to the whole in some way - Benoit Mandelbrot

...to use fractal without a pedantic definition, to use fractal dimension as a generic term applicable to all the variants - Benoit Mandelbrot

Barnsley Fern

  • Made to resemble the black spleenwort fern (Asplenium adiantum-nigrum), this is first described by British mathematician Michael Barnsley in his book Fractals Everywhere as an example of an iterated function system.

Sierpinski Triangle

  • First described by Polish mathematician Wacław Sierpiński in 1915, though the pattern itself was observed earlier.

  • We've seen approximations of this fractal before in Wolfram's CA (rule 90), Conway's Game of Life and Pascal's Triangle. And there are other examples.

    The Chaos Game

  • Start with three random points. The vertices of a triangle.
  • Pick another random point as your current position.
  • Select one of the triangle vertices at random.
  • Move half way between the current location and that vertex.
  • Select another vertex at random and repeat.

Lorenz Attractor

  • A simplified mathematical model for atmospheric convection developed by Edward Lorenz in 1963.

  • Lorenz also coined the phrase "butterfly effect" which describes a chaotic system's sensitivity to initial conditions. And inspired at least one really bad movie.
  • The three differential equations known as the Lorenz Equations.

  • sigma
  • rho
  • beta

Mandelbrot Set

Named after mathematician Benoit Mandelbrot

The Mandelbrot set is the set of complex numbers c for which the function fc(z) = z2 + c does not diverge when iterated from z = 0, remains bounded in absolute value.

- Wikipedia

What is a complex number and how do you square it?

A complex number has two parts. Real (a) and imaginary (b)

  • To calculate z2 + c:

  • Start by squaring z
  • (a+bi)*(a+bi)
  • a2 + abi + abi + b2i2
  • a2 + 2abi - b2
  • a2-b2 + 2abi

  • Then add c (which is also a + bi)
  • a2-b2+ca + 2abi+cb

Julia Set

  • Named after the French mathematicians Gaston Julia and Pierre Fatou.

  • Very closely tied to the Mandelbrot set. Then math and implementations are almost identical.

  • In the Julia set, c is constant instead of changing for each point.
